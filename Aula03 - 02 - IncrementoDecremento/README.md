# Learning about increment and decrement

- It is a way of treating values in variables in a simplified way.

## Shapes

- pre increment or pre decrement
- post increment or post decrement
- compact
- didactic

## Usage Examples

    age++ //age = age + 1;
	age+=2; // age = age +2
	age*=2; // age = age * 2
	age/=2; // age = age / 2
	age--; //age = age - 1;
	age-=10; // age = age - 10;
